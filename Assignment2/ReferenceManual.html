<a>

Abstract

The anchor tag is used to define a hyperlink, often used to create a link from one page to another. The anchor tag can be used within any other tag to make that text or image into a hyperlink. The anchor tag can even be used to create links on it's own page, known as a bookmark. The default style of the anchor tag is to have an underline with a link that has not been visited before showing blue, and purple for pages that have been visited.

Details

  • href: Used to create a hyperlink to a specified page.
  • download: Used to start a download of a file on the server.
  • hreflang: The language of the linked document.
  • ping: Used by developers to track when a user navigates to a page.
  • rel: Explains a relationship between the current document and the one that is hyperlinked.
  • target: Exlpains where to open the link document (ex. new tab, new window).
  • type: The type of media that is linked by the hyperlink.

Examples

<a href="https://google.com">Google</a> Google
<a href="https://google.com" target="_blank">Google in new tab</a> Google in new tab
<a href="test.txt" download="test.txt">Download test.txt</a> Download test.txt
